# 2. 로케이션 기반 리스크 분석  
: 작업자 개인의 숙련도에 따른 오류만이 아닌, 
업무의 환경적인 요인으로 인해 발생되는 오류가 있을 수 있다.
  
해당 분석에서는  
- 상단/하단 단수 여부
- 좌/우 끝 로케이션 여부
- 특정 위치 편중 현상
  
를 기준으로 오류발생과의 관련성 패턴을 분석하여,  
작업 현장적 리스크를 알아내고, 작업 현장의 환경적 요인으로인한 오류발생에 대한 개선방향을 도출하고자 하였다.
