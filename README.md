# 재고조사 작업자의 생산성 및 오류율 분석을 통한 문제점 및 원인파악

## 프로젝트 개요
물류센터 재고 검수 과정에서 발생하는 작업 오류 데이터를 분석하여  
작업자별 오류율, 재고 불일치 패턴, 작업 효율성을 분석한 프로젝트입니다.

본 프로젝트에서는 현장의 SBC와 CC 결과 데이터를 유사하게 직접 생성하여, 실제 현장에서 도출 할 수 있는 작업자별 KPI를 설계하고 
통합 분석하여 작업자별 생산성과 오류 책임의 로직을 구현했습니다.

## 주요 분석 내용

### 1. 작업자별 오류율 분석
본 프로젝트는 물류센터 재고조사 작업 데이터를 기반으로:

- 작업자별 생산성
- 작업자별 오류율
- 작업 품질
- 작업 효율성
- 과부하 여부
- 교육 필요 대상

을 분석하기 위해 SQL 기반 KPI 분석 시스템을 구축한 프로젝트이다.

데이터는 다음 두 작업 프로세스를 기반으로 분석하였다.
## SQL 분석 

📊 1. 분석 개요
📌 분석 목적
작업자별 성과(UPH)와 오류율을 동시에 분석
고성과자 및 개선 필요 인력 식별
휴먼에러 패턴 도출 및 운영 개선 방향 제시

| 지표          |       의미               |
| ------------- | ------------------------- |
| 총 작업량      | 작업자가 처리한 총 물량     |
| 총 소요시간    | 작업에 소요된 총 시간       |
| UPH           | 시간당 처리량 (생산성 지표) |
| 전체 작업 건수 | 작업 단위 기준 건수         |
| 총 오류 건수   | 오류 발생 횟수             |
| 오류율         | 오류건수 / 작업건수        |






## 주요 인사이트

- 특정 작업자의 오류율이 평균보다 높게 나타남
- 특정 로케이션에서 반복적인 재고 오류 발생
- 작업량 증가 시 오류율 상승 경향 발견
- 검수 프로세스 개선 필요성 확인


### 2. 작업오류 분석
( 실제 재고와 시스템 재고 불일치 분석 )

분석 포인트:
- 불일치 발생 빈도
- 반복 발생 패턴
- 재고 불일치 원인 파악
    - 로케이션별 오류 집중 구간 분석(- 주요 오류 로케이션)


### 3. 작업자별 생산성 측정 및 효율 분석
작업 시간 기반 효율 분석

분석 포인트:
- 평균 작업 시간
- 작업량 대비 오류율
- 작업 속도 비교


## 4.- 작업 효율 개선 인사이트 도출
  - 작업자별 오류율 측정 및 분석
     - 고성과자 식별
     - 개선 대상 식별
     - 휴먼에러 패턴 분석
